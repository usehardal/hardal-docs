## Data Privacy

We care about your and your users data privacy. So we don't store any users cookie, Wallet ID's or PII data.

### Data Retention

Feel free to send any request to us for your data retention or deletion progress. As default, we can use customers and users data retention up to 2 years.

### Data Deletion Request

- You can submit a data deletion request using [this](https://app.usehardal.com/data-deletion) form.
