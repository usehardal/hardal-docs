# Hardal

*Hardal* is designed to help you understand and improve the user experience on your website by collecting cookieless and privacy-first and analyzing a variety of UX metrics on **web2** and **web3** ecosystem. 

This documentation will guide you through the process of setting up and using the tool, as well as provide best practices and tips for getting the most out of it. Whether you are a beginner or an experienced user, we hope that this documentation will help you make informed decisions about how to optimize your website and enhance the user experience. 

Thank you for choosing Hardal, and we look forward to helping you succeed.

## What is Hardal?

*Hardal* (Turkish word for "mustard", pronounced /hardal/, like "hard" and "al") is a simple web analytics tool that helps you collect *Web3* metrics.

## Why you should  use Hardal?

Here is the some great features:

- Cookieless
- Privacy-first
- Lightweight
- Free
- Custom events and segmentation
- API Integration
